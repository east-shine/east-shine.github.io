

2.0.1 업데이트 이후 문제
--
스마트공시를 이용해주시는 유저분들께 안내의 말씀 드립니다.

최근 공시알림 기능에 문제가 있어서 원인을 살펴보니 알림을 구독하는 토픽의 허용량이 초과하는것으로 밝혀졌습니다.

<U>**실제 에러 메시지 내용**</U>
```json
{
  "error": {
    "code": 429,
    "message": "Topic quota exceeded.",
    "status": "RESOURCE_EXHAUSTED",
    "details": [
      {
        "@type": "type.googleapis.com/google.firebase.fcm.v1.FcmError",
        "errorCode": "QUOTA_EXCEEDED"
      }
    ]
  }
}
```
스마트공시는 개인이 개발한 앱으로 현재까지 무료로 운영을 해오고 있으며 앞으로도 **계속 무료**로 운영될 것입니다.
하지만 현재의 상태로는 푸시알림을 안정적으로 발송할 수 없습니다. 

현재 구글의 Firebase Cloud Messaging(FCM)의 [토픽 메세징]((https://firebase.google.com/docs/cloud-messaging/android/topic-messaging))을 사용하고 있는데 위에서 언급한 에러때문에 더 이상 이 방식은 사용할 수 없을 거 같습니다.

대신 사용자를 기준으로 기기에 직접 푸시 메시지를 발송하는 방식으로 전환할 예정인데 사용자마다 설정한 관심기업, 관심 보고서 및 키워드를 필터링하여 발송하기 위한 서버의 구축이 추가적으로 필요합니다.

현재 활성화 유저가 1700여 명 정도인데 이 정도 유저의 광고 수익만으로 서버를 운영하는 일이 쉽지는 않습니다.
주변에 홍보도 해주시고 광도도 자주 클릭해 주시면 운영하는데 많은 도움이 됩니다. (너무 많이 클릭하면 어뷰징으로 간주되니 적당히 부탁드립니다.)
